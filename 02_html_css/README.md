# Login & Register Page

This project is a basic **login and register UI** created using **HTML and CSS** only.  

This project not using JavaScript. The switching between Login and Register is done using **CSS radio buttons logic**.

---

## What this project do

- Show login form
- Show register form
- Switch between forms without JS
- Responsive for all screen size
- Simple layout for beginners

---

## Basic Working Process (How it works)

1. Page is divided into **two sections**
   - Left side: Logo and platform information
   - Right side: Login and Register form

2. Two radio buttons are used (hidden)
   - One for Login
   - One for Register

3. When user click on **Login tab**
   - Login radio button get checked
   - Login form is shown using CSS

4. When user click on **Register tab**
   - Register radio button get checked
   - Register form is shown

5. CSS `:checked` selector control which form is visible

This is simple logic and good for basic understanding.
